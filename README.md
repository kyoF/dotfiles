# How To Use
```
設定 > セキュリティとプライバシー > フルディスクアクセス のターミナルにチェック
```
```
xcode-select --install
git clone https://github.com/kyoF/dotfiles.git
cp .env.sample .env
-- edit .env file --
script/initial.sh
```

# Manually Install
* Xcode
* LINE

* 設定
  * 設定 > セキュリティとプライバシー > フルディスクアクセス の Alacritty にチェック
  * 設定 > 通知と集中モード > 通知が必要ない項目の通知をOFF
